conkw
Welcome to conkw. There are a few things you can do from there:
  • Go to the grabber reference. Grabbers are programs that gather data. Either from your computer, form the internet, from anywhere, really. Thay are what makes conkw so useful. There are already a few grabbers in the default install, so click above to go check them out.
  • Go to the documentation. A great place to learn all about conkw. From basic usage and configuration, reference, up to the most advanced customization, it's all there.
  • Go to the GitHub. All the code, the documentation, the issues, well, the GitHub.
  • Go to the customized UI. This is your UI. It has been initialized with a few things at installation time, but it's the place for you to build your customized dashboards. The files are in the ui/ directory of the installation folder.
