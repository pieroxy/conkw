Documentation
  • Note that this grabber only works on Linux and other systems where the lm-sensors program is available.
  • This UI shows a small sample of the metrics available through the lm-sensors command-line program.
  • If you don't see anything here, look in the system.log file for an error message. It is likely you need to install the program on your system. On Ubuntu it can be done with sudo apt install lm-sensors
  • There are a lot more metrics available that this UI doesn't show.
  • If the list of metrics below looks like a mess it's because it is. lm-sensors output metrics as it discovers them. ConkW just grabs them and exposes them through the API.
  • Look at the documentation for an explanation of the naming convention used for the metrics.
