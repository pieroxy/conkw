This is a sample ExternalMetricsGrabber named "test_emi". In order to ingest data to it, just execute this simple curl request:

curl 'http://localhost:12789/emi?ns=test_emi&name=metric1&value=2&type=num' -X POST

and watch as this page update itself. Now you can try and populate the other three metrics.
