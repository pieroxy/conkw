Documentation
  • Until you have configured an account in the config.jsonc file, you won't see anything in there.
  • The configuration is pretty simple: add a property called imapConf0 and as a value "name:server:port:login:password" where "name" is a human readable name for the account.
  • Be aware that depending on the number of unread emails you have, the first fetch can take a while (one or more minutes)
  • If you configure several accounts, all the emails will be merged. If you want them separate, use several instances of this grabber in the config file.
